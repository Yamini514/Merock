Flow 1 — Enquiry → Deal (core business pipeline)

Exact tables touched as an enquiry travels: enquired → visited → negotiating → converted.

flowchart LR
    A[Client submits
enquiry form] --> B{{"INSERT enquiries
(status='enquired')"}}
    B --> C{{"auto-assign agent_id
(property's agent)"}}
    C --> D{{"INSERT notifications
to agent"}}
    D --> E[Agent contacts client
INSERT enquiry_activities]
    E --> F[Schedule visit
INSERT property_visits]
    F --> G{{"UPDATE enquiries
status='visited'"}}
    G --> H{Client decision}
    H -- negotiate --> I{{"status='negotiating'
+ activities timeline"}}
    H -- not interested --> J{{"status='closed'
+ closed_reason"}}
    I --> K[Agent records deal
INSERT transactions]
    K --> L{{"UPDATE enquiries
status='converted'"}}
    L --> M{{"UPDATE properties
status='reserved'"}}
    M --> N[Token payment
INSERT payments]
    N --> O{{"webhook: payments.status='success'
transactions.status='agreement'"}}
    O --> P{{"completion:
transactions.status='completed'
properties.status='sold'/'rented'"}}
    P --> Q{{"referral check:
referrals.status='converted'
reward_amount set"}}
    P --> R{{"INSERT notifications
client + agent + admin"}}

Flow 2 — Authentication & Token Lifecycle

Login, JWT + rotating refresh tokens, reuse detection.

flowchart LR
    A[Login: email + password] --> B{{"SELECT users
verify password_hash"}}
    B -- fail --> C{{"INSERT audit_logs
action='login_failed'"}}
    B -- ok --> D{{"INSERT auth_tokens
purpose='refresh' (hashed)"}}
    D --> E["JWT access token (15 min)
+ refresh cookie"]
    E --> F[Access token expires]
    F --> G{{"match auth_tokens.token_hash
not expired / not used"}}
    G -- valid --> H{{"rotate: mark used_at,
INSERT new token"}}
    G -- reused --> I{{"revoke all user tokens
INSERT audit_logs"}}
    H --> E

Flow 3 — Payment (gateway round-trip, idempotent)

A payments row exists before the gateway call; webhook only flips status.

flowchart LR
    A[Client pays token/rent] --> B{{"INSERT payments
status='pending'
idempotency_key UNIQUE"}}
    B --> C[Create gateway order
store gateway_ref]
    C --> D([Razorpay / Stripe])
    D -- webhook --> E{verify signature}
    E -- duplicate key --> F[ignore — already processed]
    E -- success --> G{{"payments.status='success'
paid_at=now()"}}
    E -- failure --> H{{"payments.status='failed'"}}
    G --> I{{"advance transactions.status
booked to agreement"}}
    I --> J{{"INSERT notifications (receipt)
INSERT audit_logs"}}
